### ОБЩИЕ ###
Shift+F9 - запомнить текущую конфигурацию
Alt+F9 - переход в полноэкранный режим и обратно
Alt+F8 - вызов списка ранее вводимых команд
Ctrl+E - вызов предыдущей команды в ком. строку
Ctrl+Ins - копирование содержимого ком. строки в буфер обмена
Shift+Ins -вставка в ком. строку содержимого буфера обмена

### ПАНЕЛИ ###
Ctrl+U - поменять панели местами
Ctrl+O - убрать обе панели
Ctrl+F1, Ctrl+F2 - по отдельности убирать и выводить каждую из панелей
F11 - временная панель (сохранить панель - Alt+Shift+F2)
Ctrl+1 ...10 - меняем вид отображения панели
Ctrl+R - обновить содержимое панели

### НАВИГАЦИЯ ###
Alt+F1, Alt+F2 - вывод доступных дисков (если для неактивной панели выбрать диск, то откроется директория, как на активной панели, аналог Alt+i для MC)
Ctrl+\ - переход в корень диска
Alt+F11 - просмотр истории просмотров и редактирования
ALT+F12 - отобразить папки которые открывались ранее

### ФАЙЛЫ/ДИРЕКТОРИИ ###
Shift+F4 - редактировать файл (создать новый файл)
Ctrl+Q - режим быстрого просмотра файлов
F3 - просмотр файла
F4 - редактирование выбранного файла
F5 - копирование
F6 - переименование
F7 - создать папку
F8 - удаление
Ins - выделить несколько файлов
Ctrl+A - изменение атрибутов файла
Shift+F1 - добавить файлы к архиву
Shift+F2 - извлечь файлы из архива
ALT+F6 - создать NTFS ссылку на папку (аналог симлинков в Unix)
SHIFT+F8 - менять кодировку файла (для режимов F3 / F4)

### ПОИСК ###
Alt+F7 - поиск файла
Alt+F10 - поиск папки
Alt удерживать и вводить имя - быстрый поиск

### ФИЛЬТРЫ/СОРТИРОВКА ###
Crtl+F12 - режим сортировки
Ctrl+I - фильтр по расширениям файлов (выбрать по клавише Пробел и затем Enter)

### СПИСОК ПРОЦЕССОВ ###
CTRL+F6 - сортировка по размеру
F8 - удалить процесс
F3 - просмотр инф. о процессе
