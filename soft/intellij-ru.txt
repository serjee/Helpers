Ctrl+Shift+Alt+S - открываем окно Структура проекта
Ctrl+Alt+S - открываем Настройки
Ctrl+Shift+A - вызов команд по имени

Ctrl+G - переход к строке по номеру
Ctrl+U - переход к суперметоду или классу
Ctrl+B - переход к объявлению
Ctrl+Alt+B - переход к реализации
Ctrl+Shift+I - быстрый просмотр определения функции
Ctrl+Shift+B - переход к объявления объекта
Alt+Up/Down - перейти к предыдущему / следующему методу
Ctrl+]/[ - перейти в конец / начало блока

Ctrl+Shift+F - поиск в проекте
Ctrl+Shift+R - замена в проекте
Ctrl+N - поиск класса по имени
Ctrl+Shift+N - поиск файла по имени
Ctrl+Shift+Alt+N - поиск символа (является быстрым способом нахождения элемента в проекте)

Ctrl+/ - однострочный комментарий / убрать комментарий
Ctrl+Shift+/ - много строчный комментарий / убрать комментарий
Ctrl+Alt+O - удаляем лишние/неиспользуемые import
Ctrl+D - дублировать строку
Ctrl+Y - удалить строку
Ctrl+O - переопределяем метод

Ctrl+Shift+]/[ - выделить код до конца / начала блока
Ctrl+Shift+F7 - выделить (отметить) используемое в файле
Ctrl+W - расширяет область выделения вокруг курсора, Ctrl+Shift+W, уменьшает
Ctrl+Alt+T - генерировать вокруг выделенного кода try/catch, if/else, while и т.д.
Alt+Insert - генерировать конструкторы к выделенным полям класса, геттеры и сеттеры

Alt+F7 - найти использования в файле (в отдельном окне)
Ctrl+F7 - найти использования в файле (выделением)
Ctrl+F12 - показать список методов класса
Ctrl+H - показать иерархии типа
Ctrl+Shift+H - показать иерархии метода
Ctrl+Alt+H - показать иерархии вызовов

Shift+F9 - запустить в debug режиме
Shift+F10 - запустить выполнение
